CORS and Cookie-parser
cors allows restricted resources on a web page to be requested from another domain outside the domain from which the first resource was served. cookie-parser is used to parse the Cookie header and populate req.cookies with an object keyed by the cookie names.

```
npm i cors cookie-parser
```

JWT- JSON Web Token (JWT) is a compact, URL-safe means of representing claims to be transferred between two parties. It is commonly used for authentication and information exchange.

